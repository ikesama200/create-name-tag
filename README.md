# 実行方法
---
## 動作環境  
- Python 3.10 以上  
---
## ダウンロード  
- リポジトリをクローンしてください  
---
## 環境構築
- Python 3.10 以上をインストールしてください
- リポジトリをダウンロード直後は以下を実行してください
### windowsの場合
    `python -m venv venv`  
    `venv\Scripts\activate`  
    `pip install -r requirements.txt`  

### macOS / Linux
    `python -m venv venv`  
    `source venv/bin/activate`  
    `pip install -r requirements.txt`  
---
## 起動
- venvを起動状態で以下のコマンドを実行してください  
    `python .\main.py`
